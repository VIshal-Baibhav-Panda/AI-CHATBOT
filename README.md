🤖 AI Chatbot — Codec Technologies Internship Project

A modern AI-powered chatbot built using Flask (Python backend) and Vanilla JavaScript frontend, integrated with OpenRouter API for real-time AI responses.

🚀 Features

💬 Real-time AI conversation

⚡ Typing animation (ChatGPT-like experience)

📚 Sidebar chat history

🗑 Delete chat functionality

➕ New chat system

💾 Local storage (chat persistence)

🎨 Modern responsive UI

🌐 API-based AI responses (LLM integration)

🛠 Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Python (Flask)

API: OpenRouter (LLMs like Llama, Mistral, etc.)

⚙️ Setup Instructions
1️⃣ Clone the repository
git clone <your-repo-link>
cd AI-CHATBOT
2️⃣ Install dependencies
cd backend
pip install -r requirements.txt
3️⃣ Add API Key

Create a .env file inside root folder:

OPENROUTER_API_KEY=your_api_key_here
4️⃣ Run Backend
python app.py
5️⃣ Run Frontend

Open frontend/public/index.html in browser
OR

Use Live Server (recommended)
